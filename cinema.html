Site de Cinema

 Projeto de site de cinema que tem por objetivo vendas de ingresso e informações sobre os filmes. Este projeto foi feito com HTML, CSS e JavaScript. Foi utilizado para esse trabalho técnicas para deixar o site responsivo, com auxílio do Bootstrap, como também foram utilizados botões, hiperlinks, menu suspenso e cards para cada filme para deixar o site mais bonito e agradável para o usuário.
